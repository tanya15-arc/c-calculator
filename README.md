# 🧮 C Calculator

A simple **Calculator application** built using the **C programming language**. This project performs basic arithmetic operations such as addition, subtraction, multiplication, division, and modulus through a menu-driven interface. It helped strengthen my understanding of programming fundamentals, operators, conditional statements, and functions in C.

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Menu-driven interface
- User-friendly command-line interaction
- Input validation for basic operations

---

## 🛠️ Technologies Used

- C Programming
- GCC Compiler (or any standard C compiler)

## ▶️ How to Run

### 1. Clone this repository

```bash
git clone https://github.com/your-username/c-calculator.git
```

### 2. Navigate to the project folder

```bash
cd c-calculator
```

### 3. Compile the program

```bash
gcc calculator.c -o calculator
```

### 4. Run the executable

**Windows**
```bash
calculator.exe
```

**Linux/macOS**
```bash
./calculator
```
